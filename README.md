# 🚢 Titanic Survival Prediction (ML + Streamlit)

A Machine Learning project that predicts whether a passenger survived the Titanic disaster using classification algorithms and a beautiful Streamlit web app.

---

## 🔥 Features

* 📊 Exploratory Data Analysis (EDA)
* 🧹 Data Cleaning & Preprocessing
* 🤖 Machine Learning Model (Naive Bayes)
* 🌐 Interactive Web App using Streamlit
* 🎨 Modern UI with animations

---

## 📁 Project Structure

TITANIC_SURVIVAL_PREDICTION/
│
├── app.py              # Streamlit Web App
├── model1.py           # Model training script
├── eda.py              # Data analysis script
├── model.pkl           # Trained model
├── titanic.csv         # Dataset
├── requirements.txt    # Dependencies
└── README.md           # Project documentation

---

## ⚙️ Installation

### 1️⃣ Clone the repository

git clone https://github.com/selvan-01/titanic-survival-prediction.git
cd titanic-survival-prediction

### 2️⃣ Install dependencies

pip install -r requirements.txt

---

## ▶️ Run the Application

streamlit run app.py

👉 Open in browser: http://localhost:8501

---

## 🧠 Machine Learning Model

* Algorithm: Gaussian Naive Bayes
* Features Used:

  * Pclass
  * Sex
  * Age
  * SibSp
  * Parch
  * Fare
  * Embarked

---

## 📊 Sample Prediction

Input:
Pclass = 3
Sex = Male
Age = 22
Fare = 7.25

Output:
❌ Did Not Survive
